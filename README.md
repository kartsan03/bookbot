# Bookbot - interactive analysis of words and characters in your favorite books

## Little bit about project

This is my first project made entirely on Python. 

I built this while learning the basics of coding on **Boot.dev**. It is a simple tool that takes a text file (like a book) and counts how many words and letters are inside.

## What it does:
* It reads a book file.
* It counts all the words.
* It counts how many times each letter appears.
* It prints a nice report for the user.

## Why Python?
I used Python because it is very easy to read and great for beginners. It handles text and "dictionaries" perfectly, which is exactly what I needed for this project.

1. Make sure you have a book file (like `frankenstein.txt`) in a folder named `books/`.
